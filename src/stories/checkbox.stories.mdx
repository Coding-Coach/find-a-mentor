# Checkbox

The `Checkbox` component is a specialized component that brings cross-browser consistency to the appearance of the Checkbox
instead of letting each browser do as they please, visually.  To accomplish this, we've wrapped some custom HTML and SVG
around a primitive `<input type="checkbox" />` element, and then hide the underlying checkbox.

With the exception of a `LabelComponent` prop, all props are forwarded to the underlying `input` primitive.
